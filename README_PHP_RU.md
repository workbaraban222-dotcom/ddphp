# DOUBLE DAMAGE PHP

Это PHP-версия текущего сайта 1в1.

## Как залить

1. Залей все файлы из этой папки на PHP-хостинг.
2. Главная страница: `index.php`.
3. Админка: `admin.php`.
4. Данные сайта лежат в `data/db.json`.
5. Для работы админки сервер должен иметь права на запись в `data/db.json`.

## Backend

- Сайт читает данные через `api/site/index.php`.
- Админка сохраняет изменения туда же.
- `.htaccess` добавлен, чтобы `/api/site` открывал `api/site/index.php`.

Если хостинг не поддерживает `.htaccess`, путь `api/site/index.php` всё равно работает напрямую.
