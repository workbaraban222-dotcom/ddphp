# DOUBLE DAMAGE PHP

Это PHP-версия сайта с рабочей общей базой для админки.

## Как залить

1. Залей все файлы из папки `DOUBLE_DAMAGE_php_final_1to1` на PHP-хостинг.
2. Главная страница: `index.php`.
3. Админка: `admin.php`.
4. Общая база сайта: `data/db.json`.
5. На сервере у файла `data/db.json` должны быть права на запись.

## Как работает backend

- Сайт читает свежие данные через `api/site/index.php`.
- Админка сохраняет изменения туда же через обычный `POST`.
- После сохранения изменения видят все посетители, потому что данные берутся из серверного `data/db.json`, а не из локального кеша браузера.
- `.htaccess` оставлен для короткого пути `/api/site`, но прямой путь `api/site/index.php` тоже работает.

## Если админка не сохраняет

Проверь на хостинге права записи:

- папка `data`: `755` или `775`;
- файл `data/db.json`: `664` или `666`, если хостинг иначе не дает писать.

Если загружаешь много больших фото, увеличь на хостинге лимит `post_max_size`, потому что картинки сохраняются в базу.
